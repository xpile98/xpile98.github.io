🔍 블로그 공정위 문구 검사기 - 페어블로그

블로그 콘텐츠 내 공정위 문구 노출 여부를 자동으로 분석합니다.

소개

페어블로그는 네이버 블로그 전용 공정위 문구 검사기입니다.
체험단 홍보글의 이미지와 텍스트를 자동으로 분석하여,
공정위 표시·광고 심사지침에 따라 정책 위반 여부를 판단해줍니다.
복잡한 광고 문구 검사를 누구나 쉽게, 빠르게 수행할 수 있도록 도와드립니다.

주요 기능

  • 도메인 기반 이미지 분석: 공정위 도메인 목록을 바탕으로 이미지 내 정책 문구 감지
  • 텍스트 문구 검사: 블로그 글의 제목과 본문에서 광고 문구 여부 판단
  • OCR 이미지 인식: 이미지에 포함된 문구까지 자동 인식해 추가 검증
  • 정책 분류 태그: 정책 지킴, 위반, 내돈내산, 불분명 등으로 자동 분류
  • 요약 보고서 생성: 검출 결과를 TXT 파일로 저장하거나 복사 가능
  • 연속 페이지 분석: 1~20페이지까지 다중 포스트 자동 검사 지원
